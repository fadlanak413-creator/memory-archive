Cara Penggunaan
1. Upload Foto — Unggah ke Google Drive / Photos, salin link "Anyone with link can view".
2. Tambah Kenangan — Buka tab "Tabel Kenangan", isi semua kolom termasuk kategori dan mood.
3. Tempel Link Foto — Paste link foto di kolom "Link Foto" saat menambahkan kenangan.
4. Filter & Cari — Gunakan filter Kategori, Mood, atau kolom pencarian untuk menemukan kenangan.
